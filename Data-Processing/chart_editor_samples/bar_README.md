# 柱状图数据表格填写说明

柱状图用于比较不同类别的数值大小

## 数据表格样例

```csv
Category,Value,Subcategory
产品A,120,Q1
产品B,180,Q1
产品C,150,Q1
产品D,200,Q1
产品E,160,Q1
```

## 列说明

### Category
- **描述**: 类别名称
- **示例**: 产品A、产品B、产品C
- **说明**: 要比较的类别，每个类别会生成一个柱子

### Value
- **描述**: 数值
- **示例**: 120、180、150等
- **说明**: 每个类别对应的数值，决定柱子的高度

### Subcategory
- **描述**: 子类别（可选，用于堆叠柱状图）
- **示例**: Q1、Q2、Q3、Q4
- **说明**: 用于创建堆叠柱状图，相同子类别的数据会堆叠在一起

## 填写提示

- Category和Value列是必需的
- Subcategory列用于创建堆叠柱状图
- 可以按数值大小自动排序
- 支持水平柱状图显示
