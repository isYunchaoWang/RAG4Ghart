# 漏斗图数据表格填写说明

漏斗图用于显示流程中各阶段的转化情况

## 数据表格样例

```csv
Stage,Value
访问,1000
注册,800
下载,600
试用,400
购买,200
```

## 列说明

### Stage
- **描述**: 阶段名称
- **示例**: 访问、注册、下载、试用、购买
- **说明**: 流程中的各个阶段，按顺序排列

### Value
- **描述**: 数值
- **示例**: 1000、800、600等
- **说明**: 每个阶段对应的数值，通常是人数或数量

## 填写提示

- Stage和Value列都是必需的
- 阶段应该按流程顺序排列
- 系统会自动计算转化率
- 数值通常递减，但也可以递增
